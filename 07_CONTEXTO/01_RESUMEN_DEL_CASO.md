# Resumen del Caso: Análisis y Visualización de la Actividad Sísmica en Ecuador

## 1. Introducción
Ecuador está ubicado en una zona tectónica de alta complejidad, debido a la subducción de la Placa de Nazca que se encuentra bajo la Placa Sudamericana; el presente proyecto utiliza datos históricos y en tiempo real del IG-EPN para facilitar la comprensión de patrones sísmicos através de herramientas modernas de software.

## 2. Problema
La cantidad de datos generados por sensores sísmicos es masiva y con frecuencia muy difícil de interpretar para usuarios no técnicos e incluso tomadores de decisiones, por lo que existe la imperiosa necesidad de centralizar, limpiar y visualizar estos datos de manera interactiva.

## 3. Objetivos
*   **Limpieza de Datos**: Procesar el catálogo sísmico nacional para eliminar inconsistencias y errores de formato.
*   **Visualización**: Implementar un dashboard interactivo utilizando Python y Streamlit para representar magnitudes, profundidades y geolocalización de eventos.
*   **Análisis Estadístico**: Identificar zonas de mayor recurrencia sísmica en el territorio ecuatoriano.

## 4. Tecnologías Utilizadas
*   **Lenguaje**: Python 3.x
*   **Framework de Visualización**: Streamlit
*   **Procesamiento de Datos**: Pandas / NumPy
*   **Fuente de Datos**: IG-EPN (Instituto Geofísico de la Escuela Politécnica Nacional)
